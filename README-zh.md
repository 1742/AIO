<h2>概述</h2>
这个mod提供了一条<b>格式塔帝国</b>的全新的化身天灾路线——<i><b>万物归一</b></i>。这是一条通过现实编辑和应用无限的技术飞升路径。通过完成<i><b>视界引擎</b></i>，熬过局势并建造专属巨构<i><b>犹格索托斯</b></i>实现飞升，主宰现实法则。
<br>
尽管在这条飞升路径上你还是会看到许多原版飞升的影子<del>（什么拼好灾）</del>，但经过作者（和AI）的反复折腾，现在它引入了一套全新的<b>定向应用无限</b>系统（Directional Apply Infinity Thesis, DAIT）。不再是枯燥的加成，而是直接修改宇宙的底层代码！<del>（不就是换种方式加buff吗……）</del>

<h2>内容</h2>
<ul>
<li>新增飞升<b>万物归一</b>与化身天灾路径</li>
<li>新增<b>定向应用无限</b>系列增益</li>
<li>新增一个起源万物归一</li>
<li>新增巨构犹格索托斯</li>
</ul>

<h2>核心玩法：法则编辑</h2>
和原版的3条灾飞路径一样，你需要先采纳三个飞升才能选择<b>万物归一</b>飞升。通过积累<b>超然逻辑</b>，你不再是简单的点科技，而是直接将<b>超然逻辑</b>转换成帝国修正。
现在的天灾Buff分为三类，你可以随时切换，根据你的<b>超然逻辑</b>产出线性（在数值较低时）增长。

<h2>流程攻略</h2>
<h3>天灾1级</h3>
采纳后解锁：
<ul>
<li><b>指数级学习理论：</b>解锁套娃脑区划（递归节点），无限建造，产科研和凝聚力。</li>
<li><b>视界引擎：</b>产出能量和暗物质，为后续做准备。<del>（依然是以太相引擎的亲戚）</del></li>
</ul>
<h3>天灾2级</h3>
完成<i><b>视界引擎</b></i>后解锁：
<ul>
<li>升级为<i><b>万门之门</b></i>。注意：<i><b>万门之门</b></i>有较高的能量维护费，而且开启局势<b>融合宇宙</b>后还有动态能量维护费。</li>
<li><b>应对建议：</b>在升级之前造几个仓库存一下<i><b>视界引擎</b></i>产的能量和暗物质，开启局势后在<b>DAIT</b>中点<b>真空零点能级</b>（加岗位能量产出）来防止经济崩盘。</li>
</ul>
<h4>融合宇宙局势</h4>
你需要花费大量的能量币和暗物质推进局势。在局势推进时，你可以通过<b>DAIT</b>来获得各种好处。
<br><br>
完成局势1阶段：
<ul>
<li>解锁<b>万门之门节点</b>，全图飞星门，无视国境（现在蠢驴可能改了巨构的建造机制，不能在无法到达的星系建造巨构，所以这个功能只能在你的工程船能到的地方建造，还有一种用法是用先弹弓把工程船弹出去……）。</li>
<li>解锁特殊星系<i><b>无名之雾</b></i>。</li>
</ul>
<h3>天灾3-5级</h3>
一路升级你的巨构：<i><b>银时之匙</b></i> -> <i><b>永恒王座</b></i> -> <i><b>犹格索托斯</b></i>。
<br>
每升一级，巨构提供的<b>超然逻辑</b>都会增多，让<b>DAIT</b>的数值有个保底。到了最后阶段，全银河的舰队绑一起都不够你打的，毕竟你是在用修改器跟他们打。
<br>
完成7阶段后即完成飞升，获得游戏胜利。如果继续游戏，你可以通过犹格索托斯任意编辑现实：
<ul>
<li>改变天体类型</li>
<li>改变其他帝国政府</li>
<li>抹除天体</li>
<li>抹除银河</li>
<li>...</li>
</ul>
同样的，由于不能在无法到达的星系建造巨构，所以有很多功能都有bug...
